# flutter_application 

A new Flutter project.

智能乒乓球运动分析与可视化系统  ITTM-AVS
Intelligent Table Tennis Movement Analysis and Visualization System    

项目描述：智能乒乓球运动分析与可视化系统是一个集成了深度学习、大模型技术和可视化分析的智能体育训练平台。该系统通过纯视觉感知技术实现对乒乓球运动员位置、动作姿态和球体轨迹的精准识别，结合文心大模型提供专业的运动分析和个性化训练建议，为运动员和爱好者提供科学的训练辅助和直观的数据可视化展示。
###该项目仅是大一学生练手项目，只完成了前端架构设计、页面开发和简要业务逻辑###
